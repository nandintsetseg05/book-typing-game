# Book Typing Game / Read & Type at the Same Time

A web app that turns any book into a typing practice session.
Upload a PDF, and the app extracts the text so you can read
and improve your typing speed simultaneously.

---

## 💡 The Idea

Why do typing practice on random sentences when you can type through
a book you actually want to read? Upload any PDF and turn it into
a fully interactive typing exercise.

---

## ✨ Features

- **PDF Upload** —> Upload any book or document as a PDF
- **Text Extraction** —> Automatic text parsing via PDF.js
- **Real-Time Typing** —> Live accuracy and speed tracking as you type
- **Progress Saving** —> Resume where you left off via IndexedDB (no backend needed)
- **Responsive Design** —> Works on desktop and tablet
- **Single Page Application** —> Fast, no page reloads

---

## 🛠 Tech Stack

**Frontend:** React, JavaScript, Tailwind CSS  
**PDF Processing:** PDF.js  
**Storage:** IndexedDB (client-side persistence)  
**Architecture:** Single Page Application (SPA)

---

## 🚀 How to Run
```bash
# Clone the repo
git clone https://github.com/nandintsetseg05/book-typing-game

# Install dependencies
npm install

# Start development server
npm run dev
```
